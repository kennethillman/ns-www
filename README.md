


# NS




## Launch this project in your local environment

Run the following commands:

``` bash
npm install
npm run dev
```
Then you can access it at [http://localhost:3000](http://localhost:3000).

## Compile and minify for production and launch the server
``` bash
npm run build
npm start
```
